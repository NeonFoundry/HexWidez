<h2>Installation and Usage</h2>

Download and unzip the HexWide exe file to a location of your choice.  The exe will run directly from the folder.

Select your game exe and resolution (optionally select field of view adjustment) and then click "Patch".

The tool will search for the default "find" hex code which is '39 8E E3 3F' (corresponding to 16:9 aspect ratio) and replaces this with your chosen resolution's hex value.

Both the "find" hex string and "replace" hex string/resolution combinations can be customised for different exe files if necessary.  There are commonly and less commonly used 16:9 hex aspect ratio codes in a dropdown for the find value, and this can also be directly edited.

It is also possible to directly modify the "find" and "replace" values of field of view hex strings.

Default settings support 21:9, 32:9 and other aspect ratios in a range of resolutions.

<h3>Disclaimer</h3>

HexWide modifies executable files by applying binary patches. Although it creates a backup before making changes, you should always create your own backup of any files you cannot easily replace.

Where possible, use HexWide only with games installed through platforms such as Steam, GOG, Epic Games Store, or similar services, as these clients can usually verify or restore original game files if required.
